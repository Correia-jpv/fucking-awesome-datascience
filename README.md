  - ID3 algorithm

### Unsupervised Learning
- Clustering
  - Hierchical clustering
  - k-means
  - Fuzzy clustering
  - Mixture models
- Dimension Reduction
  - Principal Component Analysis (PCA)
  - t-SNE
- Neural Networks
- Self-organizing map

- Clustering
